## 多远程仓库管理

本仓库配置了两个远程源：
- origin: 主仓库
- gitlab: 镜像仓库

### 日常操作指南

1. 推送到双仓库
```bash
# 推送到主仓库
git push origin 当前分支名

# 同步到镜像仓库
git push gitlab 当前分支名
```
